Style Guide

Colors: The colors of the Argentine flag (light blue, yellow, white, and black)

Layout Style: Words used are short descriptions, and labels for pictures. Every page should have large heading up top, maybe a short description of the page below, and then photos and words. Photos should be symmetrical, maybe 2-4 per row, with words incorporated as described above.

Website Skeleton: A main page, branches into 5 links for the 5 topics. The each topic should have brief information, then some branch out into subtopics, the topics with their own pages.
